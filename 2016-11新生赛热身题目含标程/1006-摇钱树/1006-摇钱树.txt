C 小明的摇钱树
Problem Description

小明有n块钱。他有一棵摇钱树，刚开始每天可以赚x元钱每天（一天一次，一次x元）。他可以花a元钱升级他的摇钱树，使得摇钱树每天可以赚y元钱。在这之后，他可以再花b元升级摇钱树，使得摇钱树可以每天赚z元钱。他可以随时升级摇钱树（在拿钱之前升级，那么当天就可以拿升级后的钱，他也可以同一天连升2级）。请输出m天后他最多能得到的钱。

Input
每一行包括7个整数n，m，x，y，z，a，b。
(0<=n,m,x,y,z,a,b<=1000)，读入直至EOF结束。

Output
输出最后的答案。每行一个数字。

Sample Input


10 3 1 2 3 3 3
10 10 1 2 3 3 3


Sample Output

13
34
